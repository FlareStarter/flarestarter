欢迎来到 **FlareStarter** 文档 —— 基于 **TanStack Start + Cloudflare Workers** 的
全栈、边缘原生 SaaS 起手模板。Clone 下来,在它之上构建你自己的产品。

这套文档本身也是模板的一部分:它内嵌在同一个 app 里,是 `/docs` 路由下的一个
[Fumadocs](https://fumadocs.dev) 站点,内容源文件在 `src/content/docs/`,随主站
**同一个 Worker** 一起部署(并非独立 app、也没有独立 Worker)。你可以把这些内容整体
替换成自己产品的文档,也可以继续用它记录你的 fork。想改文档本身,见
[编写文档](/docs/customization/authoring)。
